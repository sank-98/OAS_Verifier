# OAS Verifier

This repository is dedicated to the OAS Verifier project, which aims to validate OpenAPI Specification documents and ensure compliance with standards.

## Overview

The OAS Verifier will help developers ensure that their OpenAPI definitions are correct and adhere to best practices.

## Features
- Validation of OpenAPI documents  
- Compliance checking against OpenAPI standards  
- Easy to use command line interface  

## Getting Started

1. Clone the repository:  
   ```bash  
   git clone https://github.com/sank-98/OAS_Verifier.git  
   ```  
2. Install dependencies:  
   ```bash  
   npm install  
   ```  
3. Run the application:  
   ```bash  
   npm start  
   ```
